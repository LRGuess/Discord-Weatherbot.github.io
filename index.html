About

Overview:
WeatherBot is a feature-packed Discord bot designed to keep you informed about the weather conditions at any location worldwide. Whether you're planning a weekend getaway, curious about the current weather in your area, or just want to stay ahead of the forecast, WeatherBot has got you covered.

Key Features:

  1. Accurate Weather Data: WeatherBot provides real-time weather information sourced from reputable weather APIs. Get up-to-date details on temperature, humidity, wind speed, and more for any location you desire.

  2. Command-Based Interaction: Interacting with WeatherBot is as easy as typing a command! Simply use the designated prefix followed by the location you're interested in, and WeatherBot will respond promptly with the latest weather details.
    Example Command: !weather New York

  3. Forecasting: Plan ahead with WeatherBot's forecasting capabilities. Retrieve the forecast for the next few days to make informed decisions based on upcoming weather conditions.

  4. User-Friendly Interface: WeatherBot is designed with simplicity in mind. The responses are clear, concise, and include the essential information you need without overwhelming you with unnecessary details.

  5. Customizable Settings: Tailor WeatherBot to your preferences. Set your default location, choose between Celsius and Fahrenheit, or even receive regular weather updates at specific times.

  6. Informative Embeds: Weather information is presented in visually appealing embeds, making it easy to read and share with others in your Discord server.

Commands

Current Weather:

Command: !weather [location]
Description: Get the current weather conditions for the specified location.

Weather Forecast:

Command: !forecast [location]
Description: Retrieve a multi-day weather forecast for the specified location.

Default Location:

Command: !setlocation [location]
Description: Set your default location for weather updates.

Unit Preference:

Command: !setunit [celsius/fahrenheit]
Description: Choose between Celsius and Fahrenheit for temperature units.

Daily Weather Update:

Command: !dailyupdate [time]
Description: Receive a daily weather update at the specified time.

Wind Information:

Command: !wind [location]
Description: Get detailed information about the wind conditions at a specific location.

Humidity Level:

Command: !humidity [location]
Description: Check the current humidity level for a given location.

Sunrise and Sunset:

Command: !sun [location]
Description: Find out the sunrise and sunset times for a particular location.

Weather Alerts:

Command: !alerts [location]
Description: Receive any weather alerts or warnings for the specified location.

Customize Response Format:

Command: !format [embed/plain]
Description: Choose between an embedded or plain text format for weather responses.

Bot Information:

Command: !weatherbotinfo
Description: Get information about WeatherBot, including version and support details.

Help Command:

Command: !help
Description: Display a list of available commands and their descriptions.
